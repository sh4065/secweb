# The Cake Shop

## Purpose
The purpose of the website is to showcase and sell a variety of cakes in an easy 
and visually appealing way, allowing users to browse products and place orders smoothly.

## Pages
- Home: The homepage shows a welcome message, simple navigation on top, and a button to browse cakes.
- Cakes: The cakes page displays a selection of available cakes with images and options to view or order them.
- Cart: The cart page shows the items the user has selected, with an order summary and a button to place the order.

## Design
I used CSS to style the layout, colors, and fonts.
The website has a clean, consistent design with a soft color scheme and clear layout.

## Navigation
The website’s navigation is simple and consistent across pages, linking the homepage, cakes page, and cart page. 
The homepage has a “Browse Cakes” button, the cakes page lets users view cakes and add them to the cart.
And the cart page includes a “Place Order” button and "go back" button.
## Challenges
One challenge I faced was ensuring all pages and links worked correctly,
which I fixed by checking the code, correcting file names, and testing the website for smooth navigation.
